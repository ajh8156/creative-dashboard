## 1. 채널 구조

### 유료 채널 목록 (Airbridge 기준)

| 채널 (Airbridge) | 표시명 | 비용 보정 | 비고 |
|---|---|---|---|
| `kakao` | 카카오 | ÷1.5 | 게런티 무상캐시 포함. raw cost = 유상+무상 토탈 |
| `naver.performance_da` | 네이버 DA | ÷1.1 × 0.851 (=×0.774) | VAT 제거 + 리베이트 14.9% |
| `naver.searchad` | 네이버 SA | ÷1.1 × 0.851 (=×0.774) | VAT 제거 + 리베이트 14.9% |
| `facebook.business` | 메타 | ×1.0 | `facebook` 채널도 여기로 합산. 리베이트 없음 |
| `google.adwords` | 구글 | ×0.955 | 리베이트 4.5% |

### 제외 채널
- `kakao_crm`, `kakao_liquid_crm`, `kakao_liquid_crm_attr`, `kakao_noti` — CRM (비광고)
- `apple.searchads` — 미운영
- `message-retention` 캠페인 — CRM 리텐션

### 카카오 비용 구조
- **토탈 매체비** = Airbridge `cost_channel` = 유상(현금) + 무상(캐시) 합계
- **유상 매체비** = 카카오모먼트 대시보드 기준 (Airbridge에서 분리 불가)
- 예산 집행 추적은 **토탈 raw cost** 기준

**보정 계수 (시기별)**:
| 기간 | 계수 | 산출 근거 |
|---|---|---|
| ~2026-02 | ÷1.5 | 무상캐시 50% 페이백 |
| 2026-03~ | ÷1.763 | 현금 리베이트 14.9% + 카카오캐시 50%. 실질비용 = raw/1.5 × 0.851 |

- 카카오캐시 50%: 현금 100원 → 플랫폼 150원 (raw cost에 캐시 포함)
- 현금 리베이트 14.9%: 현금 100원 중 14.9원 환급 → 실지출 85.1원
- 계수: 1.5 ÷ 0.851 = 1.763

### 네이버 비용 구조
- Airbridge raw cost는 **VAT 포함** 금액
- 보정: raw ÷ 1.1 (VAT 제거) × 0.851 (리베이트 14.9%) = **×0.774**
- DA/SA 동일 적용
- SA 예산 집행 추적은 **토탈 raw cost** 기준 (네이버+구글 합산 2,000만/월)

### 비용 보정 최종 정리

**매체비 100만원 기준 → 실질비용 환산 예시:**

| 채널 | 매체비(raw) | VAT 제거 | 리베이트 | 캐시 보정 | **실질비용** | **보정 계수** |
|---|---|---|---|---|---|---|
| **카카오 (~2월)** | 100만 | - | - | ÷1.5 (캐시50%) | **66.7만** | ×0.667 |
| **카카오 (3월~)** | 100만 | - | 14.9% | ÷1.5 (캐시50%) | **56.7만** | ×0.567 (÷1.763) |
| **네이버 DA/SA** | 100만 | ÷1.1→90.9만 | 14.9% | - | **77.4만** | ×0.774 |
| **메타** | 100만 | - | - | - | **100만** | ×1.0 |
| **구글** | 100만 | - | 4.5% | - | **95.5만** | ×0.955 |
| **토스** | 100만 | - | 18% | - | **82만** | ×0.82 |
| **크리테오** | 100만 | - | 14% | - | **86만** | ×0.86 |
| **당근** | 100만 | - | 14% | - | **86만** | ×0.86 |
| **틱톡** | 100만 | - | 4% | - | **96만** | ×0.96 |

- 에이전시 수수료, 환율 마크업 별도 없음
- 카카오 캐시 50%: 현금 100원 → 플랫폼 150원 표시 (raw cost에 캐시 포함)
- 네이버만 VAT 포함 금액으로 Airbridge에 잡힘

## 2. 데이터 기준

### 어트리뷰션
- **기준 도구**: Airbridge (의사결정 기준)
- **어트리뷰션 윈도우**: 7일 (터치포인트 기준)
- **타임스탬프**: `touchpoint_date` 기준 (스크립트에서 `eventTimestampSource: "touchpoint_date"`)
- **실시간 모니터링만** 카카오모먼트 대시보드 참조 (Airbridge 보정 ROAS가 공식 기준)

### 매출(거래액) 정의
- Airbridge `app_web_revenue` = 주문 금액 (결제 금액 아님)
- 취소/반품 미반영 (개발팀 확인 중)
- MySQL `orders_order.payment_amount`와 차이 존재 (Airbridge 28% 과소 → 어트리뷰션 미귀속분)

### 리베이트 적용 시기
- **카카오**: 2026-03~ 현금 리베이트 14.9% 시작 (2월까지는 캐시만)
- **네이버**: 리베이트 14.9% 상시 적용
- **캐시 비율 (카카오)**: 2026년 1~6월 50% 고정

## 3. 캠페인 유형 분류

### 분류 규칙 (순서 중요)

```
1. 채널 = naver.searchad → SA
2. 채널 = google.adwords & 캠페인명에 "search" → SA
3. 캠페인명에 "appinstall" 또는 "apppurchase" → APP
4. 캠페인명에 "pbtd" → PBTD
5. 캠페인명에 "retarget" 또는 "catalog" → RT
6. 나머지 → UA
7. 캠페인명에 "sel" → 셀렉션
```

**주의사항**:
- `pbtd` 체크가 `retarget`보다 먼저여야 함 (pbtd 캠페인명에 'retarget' 포함 가능)
- `catalog` 캠페인은 UA가 아닌 RT로 분류 (구매자 분석 결과 65~78%가 기존유저)
  - kakao: `kakaoDisplay_conversion_catalog` — 구매자 77.8% 기존유저(90일+)
  - meta: `catalog_da_pr-purchase` 등 — 구매자 65~67% 기존유저

### 유형별 특성

| 유형 | 목적 | 주요 채널 | 특성 |
|---|---|---|---|
| **UA** | 신규 유저 획득 | 카카오, 네이버DA, 메타 | 기획전/셀렉션 소재. 볼륨 큼 |
| **RT** | 리타겟팅 | 카카오, 메타, 네이버DA | 카탈로그+비보 중심. 기존유저 재구매 |
| **PBTD** | 직매입/PB 상품 | 카카오, 네이버DA | 마진룸 활용. 독립 운영 |
| **APP** | 앱설치 캠페인 | 메타, 구글 | 앱그로스. 2월 재개 |
| **SA** | 검색 광고 | 네이버SA, 구글SA | 브랜드검색+키워드 |

## 6. 채널별 운영 상세

### 카카오모먼트
- **예산**: 일 1,100~2,000만원 (게런티 연간 72억, 무상캐시 70% 페이백)
- **주력 지면**: 비즈보드 (채팅탭 상단). 디스플레이는 12/15 업데이트 후 성과 저조
- **캠페인 구성**: UA(기획전/셀렉션) + RT(카탈로그/비보) + PBTD(직매입)
- **3개월 예산 추이**: RT 비중 하락 (80%→62%), PBTD 성장 (0%→21%)
- **운영 포인트**: 소재 피로도 관리가 핵심. 후심사 권한 보유. CAPI 연동 추진 중
- **담당**: 기획전→카이/엘린, 셀렉션→찬/크리스

### 네이버 DA
- **예산**: 일 500~1,200만원. 피드/스마트채널
- **특성**: 단일랜딩(PDP) 효율 좋음. 검수 까다로움
- **담당**: 찬/크리스

### 네이버 SA
- **예산**: 월 ~2,000만원 목표 (일 ~80만원)
- **특성**: 브랜드검색 + 키워드. CTR 높고 ROAS 1,200%+
- **담당**: 찬/크리스

### 메타 (facebook.business)
- **예산**: 일 430~450만원
- **캠페인**: DA(카탈로그+기획전 소구) + APP(앱설치)
- **특성**: 머신러닝 안정성 관리 중요. LF 카탈로그 중심
- **담당**: 제이슨·리아/체니 (그로스엔진)

### 구글
- **캠페인**: APP(앱설치) + SA(검색)
- **특성**: APP은 UAC 기반, SA는 소규모 (~월 150만)
- **담당**: 제이슨·리아/체니

## 8. 주요 인사이트 (분석 기반)

### UA 캠페인 구매자 특성 (2026-02-19 분석)
- 카카오 UA 구매자의 40~67%가 **가입신규** (기존 가설과 반대)
- 가입신규 CVR이 기존 구매자 대비 3~4배 높음
- → UA 효율 측정 시 신규/기존 세분화 필수. 단순 ROAS보다 신규유저 획득 관점 중요

### 카탈로그 캠페인 = RT (2026-02-22 분석)
- kakao catalog 구매자 77.8% 기존유저 (90일+)
- meta catalog 구매자 65~67% 기존유저
- → catalog 키워드는 RT로 분류하는 것이 실제 유저 행동에 부합

### 카카오 예산 구조 변화 (3개월 추이)
- 12월: RT 80% / UA 20%
- 1월: RT 65% / UA 18% / PBTD 17%
- 2월: RT 62% / UA 17% / PBTD 21%
- → PBTD 비중 빠르게 성장. RT 의존도 감소 추세
