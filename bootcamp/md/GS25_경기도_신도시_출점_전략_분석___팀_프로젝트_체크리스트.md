### 1-3. 배후인구 산출 공식

> ⚠️ 파일 배포 후 팀 전체 합의 — 페르소나 가중치는 EDA 결과 이후 조정 가능

```
총배후인구 = 거주인구
           + (직장인구 × 가중치)
           + (학생인구 × 가중치)
           + (유동인구 × 가중치)
```

- 가중치 상세: 자료 배포 후 팀 합의로 확정
- 학교급별(초/중/고) 세분화 가중치 별도 적용
- 페르소나별 가중치 조정: EDA 완료 후 확정

### 1-4. 페르소나 설정 방향

> EDA 결과를 보고 확정. 현재는 초기 방향만 설정.

| 페르소나 | 기반 데이터 | 초기 방향 |
|---|---|---|
| 2040 직장인 | 직장인구 | 주력 페르소나 — 소비력 높음 |
| Z세대·학생 | 학생수 | 보조 페르소나 — GS25 브랜드 친화도 높음 |
| 주거 가족 | 거주인구 | 기본 배후 |
| 야간 유동 | 버스정류장 | 지역 특성에 따라 선택 적용 |

### 1-5. 핵심 가설

> 데이터 확인 후 수정 가능

```
가설 1. GS25 미입점 + 배후인구 상위 행정동이 최우선 출점지다
가설 2. 2040 직장인 비율이 높은 행정동일수록 GS25 PB상품 소비 친화도가 높다
가설 3. 경쟁사 점포가 있는 행정동은 이미 수요가 실증된 지역이다
가설 4. 페르소나 특성이 명확한 행정동일수록 특화 PB상품 전략이 효과적이다
```

## STEP 3. 심화 분석

> 💡 EDA 결과 및 페르소나 확정 이후 진행

### 3-1. 핵심 분석 질문

```
Q1. GS25 미입점 + 배후인구 상위 행정동은 어디인가?
Q2. 경쟁사가 이미 입점한 곳 중 GS25 공백이 큰 곳은?
Q3. 2040 직장인 비율이 높으면서 GS25 버거지수가 낮은 행정동은?
Q4. 신도시별 1순위 출점 후보지와 전략 방향은?
Q5. ML 기반 1년 예상 매출 상위 행정동은 어디인가?
Q6. 출점 성공을 결정하는 핵심 변수는 무엇인가? (feature importance)
Q7. 선별된 지역의 페르소나에 최적화된 PB상품 조합은?
```

### 3-2. 심화 분석 항목

**① 기회 매트릭스**
- [ ] 경쟁강도(X) × 점포당배후인구(Y) 4분면 분류
- [ ] GS25 미입점 + 우수 등급 행정동 우선순위 도출
- [ ] 신도시별 1순위 출점지 선정 및 전략 방향 작성

**② ML — 1년 예상 매출 예측**
- [ ] Feature 정의: 배후인구 / 직장인구 / 2040비율 / 경쟁사 점포수 / 버거지수 / 페르소나 점수 등
- [ ] 타깃 변수: 점포당 연매출 추정 (외부 벤치마크 기준)
- [ ] 모델 학습 및 비교 (RandomForest / XGBoost)
- [ ] 행정동별 1년 예상 매출 산출 및 순위화
- [ ] 출점 우선순위 TOP 10 확정

**③ ML — Feature Importance**
- [ ] 변수 중요도 산출 및 시각화
- [ ] 중요 변수 기반 가중치 재검토

**④ PB상품 제안**
- [ ] 출점 우선순위 상위 행정동별 주력 페르소나 확정
- [ ] 페르소나 × PB상품 카테고리 매핑
- [ ] 지역별 특화 PB상품 조합 제안 (최소 3개 지역)
